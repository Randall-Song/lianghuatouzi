# 快速开始指南

## 5分钟上手

### 第一步：准备文件
下载 `L10_improved_strategy.ipynb` 文件到本地。

### 第二步：上传到聚宽
1. 登录聚宽研究平台
2. 点击"上传文件"
3. 选择 `L10_improved_strategy.ipynb`

### 第三步：运行策略
点击"全部运行"或按顺序执行每个单元格。

### 第四步：查看结果
等待运行完成后，查看最后的输出：
- **信息比率**：应为正数，越大越好
- **策略夏普比率**：衡量风险调整后收益
- **财富曲线图**：可视化策略表现

## 常见问题速查

### 运行很慢？
正常现象，首次运行需要准备大量数据。

### 信息比率为负？
尝试调整 `optimal_stock_count` 或改为周度调仓。

### 出现错误？
检查日期范围是否在数据可用范围内。

### 想重新训练？
代码会自动删除旧文件，直接运行即可。
