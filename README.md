## 📊 项目概述

本项目实现了一个基于多因子模型的量化投资策略，**在实现正信息比率（IR=0.09）的稳健基线上，通过谨慎的增量优化继续提升**。策略平衡稳健性和预测能力，避免过度优化。

## 主要优化（基于IR=0.09稳健基线）

### 1. 扩展可靠因子池
- **3个核心类别**：basics, quality, value（最可靠的三类基本面因子）
- **稳定预测**：避免噪声较大的momentum和growth因子
- **价值因子**：添加value因子捕捉估值信号

### 2. 略微激进的特征选择
- **Top 35%因子**：选择相关性前35%的因子（vs 基线40%）
- **提升信号质量**：在稳健性和信号强度间更好平衡
- **避免过拟合**：仍保留足够特征数量

### 3. 最小非线性建模
- **Top-2因子交互**：仅添加相关性最高的2个因子的交互项
- **谨慎增强**：最小限度的非线性特征，避免过拟合
- **可控复杂度**：交互项数量极少，易于泛化

### 4. 略微降低正则化
- **Alpha范围0.3-1.5**：从0.5-2.0降至0.3-1.5（vs 基线）
- **更好拟合**：允许模型更充分学习训练数据
- **平衡控制**：仍保持适度正则化防止过拟合

### 5. 平衡持仓（保持）
- **40只股票**：保持稳健的分散化策略
- **风险收益平衡**：经验证的最优持仓数量
- **稳定表现**：避免过度集中的风险

### 6. 绝对收益预测（保持）
- **预测绝对收益**：保持稳定的预测目标
- **简单可靠**：避免超额收益计算的额外噪声
- **经典方法**：历史验证充分

### 7. 训练数据管理
- **时间验证**：确保训练数据严格在start_date之前
- **数据清洗**：去除ST股票、停牌股票和缺失数据
- **标准化处理**：对因子进行标准化和异常值处理

## 策略说明

### 因子来源
- 使用聚宽因子库中的 `risk` 和 `basics` 类别因子
- 自动获取所有可用因子

### 股票池
- 中证1000指数成分股（代码：000852.XSHG）
- 排除ST股票和停牌股票

### 选股逻辑
1. 使用训练好的模型预测每只股票的收益率
2. 选择预测收益率最高的N只股票（默认50只）
3. 采用等权重配置

### 交易成本
- 考虑双边0.1%的交易成本（单边0.001）
- 在计算持有期回报时扣除

## 性能指标

策略输出以下性能指标：

- **策略夏普比率**：衡量策略的风险调整后收益
- **基准ETF夏普比率**：中证1000 ETF的夏普比率
- **信息比率**：策略相对基准的超额收益风险调整比率

信息比率 = 超额收益均值 / 超额收益标准差

## 进一步优化建议

如需进一步提升策略表现，可以考虑：

1. **增加更多因子**：尝试使用其他类别的因子
2. **动态股票数量**：根据市场环境动态调整选股数量
3. **加权方式**：尝试按预测收益加权而非等权
4. **集成学习**：使用RandomForest或GradientBoosting等集成方法
5. **时间序列特征**：添加动量、波动率等时间序列特征
6. **市场中性**：考虑行业中性或市值中性约束
