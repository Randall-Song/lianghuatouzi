## 📊 项目概述

本项目实现了一个基于多因子模型的量化投资策略，**核心目标是实现稳健的正信息比率**。策略采用简洁可靠的方法，避免过度优化和过拟合。

## 主要策略特点

### 1. 简化因子池
- **2个核心类别**：仅使用basics和quality（最可靠的基本面和质量因子）
- **避免噪声**：不使用动量和成长等高噪声因子
- **稳定预测**：基本面因子提供更稳定的长期信号

### 2. 温和特征选择
- **Top 40%因子**：选择相关性前40%的因子（平衡信号和鲁棒性）
- **避免过拟合**：不过度激进的选择，保留足够的信息
- **稳健性优先**：优先考虑模型的泛化能力

### 3. 简单线性模型
- **无复杂交互**：不使用因子交互项和多项式特征
- **线性关系**：专注于捕捉线性关系，避免过拟合
- **可解释性强**：简单模型更易理解和调试

### 4. 适度正则化
- **中等alpha值**：使用0.5-2.0的alpha范围（Ridge）
- **平衡偏差方差**：在拟合训练数据和泛化能力之间取得平衡
- **稳定系数**：避免系数过大导致的不稳定

### 5. 平衡持仓
- **40只股票**：提供足够分散化，降低特异风险
- **风险收益平衡**：在集中度和风险之间找到平衡点
- **稳定表现**：避免极端集中带来的高波动

### 6. 绝对收益预测
- **预测绝对收益**：直接预测股票收益率（更稳定）
- **避免基准噪声**：不依赖超额收益计算，减少误差
- **简单可靠**：经典方法，历史验证充分

### 7. 训练数据管理
- **时间验证**：确保训练数据严格在start_date之前
- **数据清洗**：去除ST股票、停牌股票和缺失数据
- **标准化处理**：对因子进行标准化和异常值处理

## 策略说明

### 因子来源
- 使用聚宽因子库中的 `risk` 和 `basics` 类别因子
- 自动获取所有可用因子

### 股票池
- 中证1000指数成分股（代码：000852.XSHG）
- 排除ST股票和停牌股票

### 选股逻辑
1. 使用训练好的模型预测每只股票的收益率
2. 选择预测收益率最高的N只股票（默认50只）
3. 采用等权重配置

### 交易成本
- 考虑双边0.1%的交易成本（单边0.001）
- 在计算持有期回报时扣除

## 性能指标

策略输出以下性能指标：

- **策略夏普比率**：衡量策略的风险调整后收益
- **基准ETF夏普比率**：中证1000 ETF的夏普比率
- **信息比率**：策略相对基准的超额收益风险调整比率

信息比率 = 超额收益均值 / 超额收益标准差

## 进一步优化建议

如需进一步提升策略表现，可以考虑：

1. **增加更多因子**：尝试使用其他类别的因子
2. **动态股票数量**：根据市场环境动态调整选股数量
3. **加权方式**：尝试按预测收益加权而非等权
4. **集成学习**：使用RandomForest或GradientBoosting等集成方法
5. **时间序列特征**：添加动量、波动率等时间序列特征
6. **市场中性**：考虑行业中性或市值中性约束
